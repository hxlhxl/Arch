# 阻塞、异步
线程遇到IO事件后
    如果等待IO事件返回，则为block
    如果不等待IO事件返回，则为非block；把IO请求发送给操作系统，继续执行下一个语句。当IO事件完成时，以事件的形式通知操作系统。
